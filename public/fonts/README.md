# Custom Fonts

Please add the following font files to this directory:

- **Lato-Black.ttf** - Used for header navigation text
- **dunbar-tall-ultra.ttf** - Used for "SITE UNDER DESTRUCTION" text

## Installation

1. Place the font files in this directory
2. Run `git add public/fonts/*.ttf`
3. Commit and push the changes

The fonts are already configured in `app/globals.css` and ready to use.
